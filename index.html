Mi Curriculum

Proyecto: Estudio fotográfico

Un estudio de fotografía necesitaba de un sitio online para mostrar su trabajo al mundo entero. Necesitaban no solo mostrar su contenido, sino un sistema para agendar todos aquellos trabajos en los que se les fuera requiriendo.

Proyecto: Sistema administrativo de tienda

Con el afan de tener un control sobre los recursos que se manejan, una tienda decidió encomendarnos la tarea de realizar el sistema acorde a su pequeña linea de productos

Proyecto: Sistema administrativo de proveedor

Para este proyecto era imprescindible tener en claro las distintas formas en las que un proveedor interactua con su clientela. Para facilitar el trabajo más laborioso de este proceso logístico, se creo este sistema.
